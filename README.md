# ☕ QR/Barcode Order Management System for Coffee Shop

[![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

An advanced order management system for coffee shops, restaurants, and logistics using QR codes and barcodes.

## ✨ Features

- ✅ Generate unique QR code for each order
- ✅ Scan QR codes and barcodes using camera
- ✅ Order status management (pending, preparing, ready, delivered, cancelled)
- ✅ Logistics tracking system with full history
- ✅ Automatic receipt printing
- ✅ SQLite database
- ✅ Customizable menu
